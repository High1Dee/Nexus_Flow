## OVERVIEW
--------
NexusFlow is a premium Bootstrap 5 landing page template designed specifically for SaaS startups, tech companies, and software product launches. The template features a modern, clean design with a focus on conversion optimization, user experience, and professional presentation.
Built with Bootstrap 5, vanilla JavaScript, and custom CSS, this template is lightweight, fast, and easy to customize.

## CUSTOMIZATION GUIDE
-------------------
### COLORS
------
The template uses CSS custom properties for colors. To change the color scheme, edit the :root variables in css/style.css:

```
:root {
    --primary: #6366f1;      /* Main brand color */
    --primary-dark: #4f46e5; /* Darker shade */
    --secondary: #10b981;    /* Accent color */
    --dark: #111827;         /* Dark text */
    --gray: #6b7280;         /* Gray text */
}
```
### FONTS
-----
The template uses Google Fonts. To change fonts:
1. Update the Google Fonts link in index.html
2. Update the font-family declarations in css/style.css

### CONTENT
-------
Replace the placeholder content in index.html with your own:
1. Update the company name and logo
2. Replace hero section content
3. Update feature descriptions
4. Change pricing information
5. Replace testimonial content
6. Update team member information
7. Modify FAQ questions and answers
8. Update footer content

### IMAGES
------
Replace the placeholder images in the images/ folder:
1. Hero dashboard image (hero-dashboard.png)
2. Feature illustrations
3. Product screenshots
4. Team member photos
5. Client logos
Use optimized images for better performance. Recommended image sizes are provided in the customization guide.

### NAVIGATION
----------
Update the navigation menu in index.html:
1. Change menu item text and links
2. Update the logo and brand name
3. Modify CTA button text if needed

### FORMS
-----
The template includes a lead capture form in the hero section. To connect it to your backend:
1. Update the form action attribute
2. Add any additional form fields needed
3. Update the JavaScript validation in main.js if necessary

### RESPONSIVE ADJUSTMENTS
----------------------
The template is fully responsive. To make adjustments:
1. Edit the media queries in css/responsive.css
2. Test on different screen sizes
3. Use Bootstrap's responsive utilities for layout adjustments

### JAVASCRIPT FUNCTIONALITY
------------------------
The template includes several JavaScript features:
1. Navbar scroll effect
2. Screenshot gallery controls
3. Pricing toggle (monthly/annual)
4. Smooth scroll for anchor links
5. Form validation
6. Scroll animations
7. FAQ accordion functionality
All JavaScript is in js/main.js and is well-commented for easy customization.
